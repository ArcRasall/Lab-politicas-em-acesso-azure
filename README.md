# Lab-politicas-em-acesso-azure

Portal de confiança do serviço consiste em uma pagina web com documentações relacionadas a leis e padrões seguidos pela microsoft para esclarecimeto aos clintes azure como certificações e leis refentes aos paises onde o serviço é fornecido.
O lab demostra os bloqueios associados ao resource group e como mover os objeto de para outro resource group para aplicar atividades que não estão habilitadas no anterior.
Criação de recurso microsoft pure view para fazer analise do seu ambiente para adequações a normas de seguranças necessarias.
Policy cria regras para os recursos e valida se os recursos criados apartir de sua aplicação estão conformes e sinaliza quais não estão de acordo, alem de sinalizar as medidas para confomização dos recursos criados.
